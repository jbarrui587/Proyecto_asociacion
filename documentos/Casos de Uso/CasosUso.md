# Casos de Uso 

## U-1: Login - Autenticación , Usuarios para dar privilegios de miembro.

## U-2: Registro -  Registrar usuario invitado en usuario miembro y así poder hacer login.

## U-3: Gestionar Usuarios: usuario admin puede crear, modificar o elminar datos de los usuarios de la base de datos.

## U-4:  Ver datos usuario: El usuario miembro y administrador pueden ver los datos de usuarios, los usuarios miembro solo pueden ver el suyo, pero el admin pueder ver todos.

## U-5: Gestionar Fotos de la galería:usuario admin puede crear, modificar o elminar datos de las fotos de la galería  de la base de datos.

## U-6: Gestionar Noticias:usuario admin puede crear, modificar o elminar datos de las noticias de la galería  de la base de datos.

## U-7: Crear comentario: Una vez logueado el usuario miembro puede poner un comentario que se mostrará en la tabla de la pestaña comentarios.

## U-8: Gestionar Comentarios: El usuario Admin puede eliminar los comentarios que crea oportuno.

## U-9: Ver contenido: Todos los tipos de usuario(invitado, miembro, admin) pueden ver el contenido de la web.
