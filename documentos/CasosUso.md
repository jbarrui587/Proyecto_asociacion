# Casos de Uso 

## U-1: Login - Autenticación , Usuarios para dar privilegios de miembro.

## U-2: Registro -  Registrar usuario invitado en usuario miembro y así poder hacer login.

## U-3: Gestionar Usuarios: usuario admin puede cambiar los datos de los usuarios y eliminar(editar) usuarios de la base de datos.

## U-4:  Ver datos usuario: El usuario miembro y administrador pueden ver los datos de usuarios, los usuarios miembro solo pueden ver el suyo, pero el admin pueder ver todos.

## U-5: Donaciones: El usuario miembro puede ir a un postal externo de paypal para realizar donaciones a la asociación.

## U-6: Ver contenido: Todos los tipos de usuario(no miembro, miembro, admin) pueden ver el contenido de la web.

## U-7: Modificar contenido de la web(noticias,eventos, actividades) solo puede hacerlo en usuario admin.

## U-8: Ver donaciones:El usuario admin puede ver las donaciones realizadas por los miembros.
